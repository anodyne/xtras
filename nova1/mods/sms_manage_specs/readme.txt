NOVA MOD - SMS STYLE MANAGE SPECS FORM

If you'd prefer that your manage specs page in Nova look more like the
SMS version of the manage specs page join form, you can install this
simple MOD that will override Nova's defaults.

INSTALLATION

1. Connect to your server with your FTP program
2. Upload manage_specs.php to application/views/_base_override/admin/pages

SUPPORT
