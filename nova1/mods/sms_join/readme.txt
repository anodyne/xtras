NOVA MOD - SMS STYLE JOIN FORM

If you'd prefer that your join form in Nova look more like the SMS version
of the join form, you can install this simple MOD that will override Nova's
defaults.

INSTALLATION

1. Connect to your server with your FTP program
2. Upload main_join_2.php to application/views/_base_override/main/pages
3. Upload main_join_js.php to application/views/_base_override/main/js
