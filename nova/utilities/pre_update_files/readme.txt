NOVA UTILITY - PRE-RELEASE UPDATE FILES

For users who installed a version of Nova prior to 1.0, you can use
these update files to get your installation from a previous version
to 1.0. Please note, updating from a pre-release version is not
supported by Anodyne.

INSTALLATION

1. Connect to your server with your FTP program
2. Delete application/assets/update
3. Upload the update folder to application/assets

SUPPORT

There is no support for this utility. It is provided as is with no
guarantees. Use at your own risk.
